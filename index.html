Las cuatro capas de cualquier elemento HTML

Content

El contenido real: texto, imagen, video. Su tamaño es el width y height que defines.

Padding

Espacio interior entre el contenido y el borde. Hereda el color de fondo del elemento.

Border

La línea que rodea el elemento. Tiene grosor, estilo y color.

Margin

Espacio exterior entre el borde y otros elementos. Siempre transparente.

justify-content — diferencias clave
space-between

El primer y último item tocan los bordes. El espacio se distribuye ENTRE ellos.

space-around

Cada item tiene el mismo espacio a sus lados. Los bordes tienen la mitad de espacio que el centro.

space-evenly

Todos los espacios son exactamente iguales, incluyendo los bordes.

center

Los items se agrupan en el centro. Combinar con gap para separarlos.

La pregunta clásica: centrar un elemento horizontal y verticalmente

La respuesta en 3 líneas

.contenedor {
  display: flex;
  justify-content: center; /* horizontal */
  align-items: center; /* vertical */
}

El truco de los ejes — nunca te confundirás

justify-content siempre va en la dirección del flex-direction.
align-items siempre va en la dirección perpendicular.

Con row: justify → horizontal · align → vertical
Con column: justify → vertical · align → horizontal
